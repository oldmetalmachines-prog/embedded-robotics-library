# python-can-j1939 (Python)

Source: https://github.com/juergenH87/python-can-j1939
Author: juergenH87
License: MIT
Language: Python

## Purpose
Full SAE J1939 stack in Python for diagnostics, logging and ECU simulation.
Supports J1939-FD (flexible data rate). Good for building diagnostic tools
for farm equipment, trucks and automotive ECUs.

## Key Folders
- `examples/` — Working examples for ECU simulation, diagnostics, address claiming
- `j1939/` — Core J1939 library

## Install
pip install python-can j1939

## Hardware Target
Any Linux machine with USB-CAN adapter or Raspberry Pi with MCP2515.
Use with SocketCAN interface.
