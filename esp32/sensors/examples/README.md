## Available Examples

1. **IMU Sensors** - Orientation and motion
   - MPU6050 - 6-axis (accel + gyro)
   - BNO055 - 9-axis with fusion

2. **Distance Sensors** - Obstacle detection
   - VL53L0X - ToF laser (2m range)
   - VL53L1X - ToF laser (4m range)

3. **Environment** - Weather/altitude
   - BMP280 - Pressure/temperature
   - BME280 - Pressure/temp/humidity

## Quick Start

Each example includes:
- Wiring diagram (ASCII art)
- Complete working code
- Dependencies in comments
- Calibration instructions
